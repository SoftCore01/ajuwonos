CGPA Calculator

I created the Cumulative Grade Point Average (CGPA) Calculator App because I realised a lot of students like myself have issues understanding and calculating their GPAs and CGPAs. With this application they no longer need to worry about the confusing calculation. All they need do is input the grade and courses into the input fields and they get the results immediately. I intend to make this a full-stack application sometime in the future. It was created using React, TailwindCSS, and Vite

Weather App

The story behind this application is an unusual one. Stay with me! While doing my internship I realized that there was usually some rainfall whenever I had to go into the office.(No, I did not go to work everyday). So there was a need to track the rainfall forecast in my city at least a day prior to going to work. Hence the Weather app was born. It was created using StitchwithGoogle for the UI design, Weather API for the weather data, React, TailwindCSS, Vite and was hosted using Vercel.
