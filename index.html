Weather App

The story behind this application is an unusual one. Stay with me! While doing my internship I realized that there was usually some rainfall whenever I had to go into the office.(No, I did not go to work everyday). So there was a need to track the rainfall forecast in my city at least a day prior to going to work. Hence the Weather app was born. It was created using StitchbyGoogle for the UI design, Weather API for the weather data, React, TailwindCSS, Vite and was hosted using Vercel
